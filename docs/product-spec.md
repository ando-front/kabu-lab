# プロダクト企画: カブラボ

## なぜこれを作るか

日本では2022年から高校で金融教育が必修化され、新NISAで「投資は特別なことじゃない」という空気が広がっている。しかし中学生向けの"本気で学べる"教材はほぼ空白地帯。既存のゲームは幼稚すぎ、大人向けツールは難しすぎる。ここに明確な機会がある。

## ターゲット

- メインユーザー: 中学1〜3年生(12〜15歳)。親が投資をしている、YouTubeで投資動画を見たことがある、数字やデータが好きといった層
- 意思決定者: 保護者。ユーザーと購入者が違うプロダクトなので、両方に響く設計が必要

## コアコンセプト

「ペーパートレードを超えた、意思決定力が育つシミュレータ」。単に仮想資金で売買するだけなら既存アプリでも十分。差別化は「なぜその判断をしたか」を言語化させ、後で振り返れる仕組み。

## MVPスコープ(現時点)

- 日本株12銘柄(中学生が名前を知っている会社中心: 任天堂、ソニー、サンリオ、ユニクロ、ディズニーなど)
- 仮想資金100万円スタートでの売買
- 実際の株価水準を起点にしたシミュレーション価格(GBMベース)
- 売買時に「なぜ」を記録するトレード日記
- ブラウザのみで完結(localStorageに保存)
- 時間の早送り機能(1x/5x/20x/一時停止)

## 設計原則

- 5分で最初の売買まで到達できること。口座開設・チュートリアルで萎えさせない
- 数字よりグラフ。PER、PBR、ROEは最初の体験には要らない
- 損しても怒られない演出。「〇〇円損しました」ではなく「この判断から何がわかった?」への変換
- 短期売買を推奨しない設計。長期保有や分散にインセンティブを置く(将来)

## 得られるベネフィット

子ども本人にとって: お金や経済ニュースを自分ごとで読めるようになる。仮説→検証→振り返りの思考サイクルが身につく。社会の仕組みへの解像度が上がる。失敗してもリアルな損失ゼロで試行錯誤できる。

保護者にとって: 学校の金融教育を補完できる。親子の会話ネタが増える。リアルマネーでの失敗リスクなく金融教育ができる。

## ロードマップ

近い順に:

1. ファーストユーザー(我が子)での検証 — どこで詰まるか、何に興奮するか観察
2. 週次振り返りレポート — 「先週の判断、正解だった? 間違ってた?」
3. ニュースイベント — 決算発表、新商品、不祥事で価格がジャンプする仕組み
4. 実データ連携 — J-Quants API経由で日次終値(認証必要、無料枠あり)
5. 保護者ダッシュボード — 子どもの学習状況を親が見られる

## 既知の制約

- 株価はシミュレーション(実際の日本株価格とは異なる)。基準価格のみ2026年4月時点の実勢を参考にしている
- 手数料や税金は現時点で未実装。教育的には重要な要素なので将来追加
- 単一ユーザーのみ(複数ユーザー管理・認証なし)
