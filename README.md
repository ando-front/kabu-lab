# カブラボ (KabuLab)

中学生向けの株式トレード体験シミュレータ。100万円の仮想マネーで、本物の日本株の値動きに近い環境で売買を学べる。

## このプロジェクトの目的

- 中学生が「市場の息遣い」を体験できる、損失リスクのない環境を作る
- 売買の判断を日記に残し、後から振り返ることで思考力を育てる
- 金融リテラシーを「実感」として身につけるための最初の入口にする

## 技術構成

- フロントエンドのみのシングルページアプリ(ブラウザだけで完結)
- バニラJavaScript、ビルド不要、フレームワーク不使用
- データはブラウザのlocalStorageに保存
- 株価は実際の水準を起点にGBM(幾何ブラウン運動)でシミュレート

ビルドツールを入れていないのは意図的で、HTMLを開けば即動く手軽さを優先している。

## 開発

```bash
# ローカルで開く
open index.html

# あるいは簡易サーバで
python3 -m http.server 8000
```

GitHub Pagesで公開する場合は `main` ブランチのルートを公開対象にするだけで動く。

## ロードマップ

- リアルっぽい値動き、12銘柄、売買、損益表示
- トレード日記、時間の早送り(1x/5x/20x)
